## Mercurial(hg) 명령어

### hg rollback
지정한 구간으로 롤백

```
hg update -C -r **revision number**
```

### hg merge

```
hg merge **branch name**
```

### hg pull

```
hg pull -b **branch name**
```

### hg branch list

```
hg branches
```

### hg current branch

```
hg branch
```
